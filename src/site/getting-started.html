Getting started with FreeSheet

FreeSheet has two components:

  • FreeSheet Core which is the core language runner
  • FreeSheet Web which helps you use the core runner in a web page and see the program in a worksheet

Installation

Easier installation is in the works, but for now it's download and build. You will need git and node.js installed.

  • Clone the two Git repositories freesheet and freesheet-webunder the same parent directory
  • Get dependencies in each one
  • Run the build in each one
  • Copy the JavaScript and CSS bundles to your project

These commands should download and build both repos:

            git clone https://github.com/FreeSheetTech/freesheet.git
            git clone https://github.com/FreeSheetTech/freesheet-web.git
            (cd freesheet-web && ./build)

Then copy the output files to your project eg:

            cp freesheet-web/dist/freesheet-all.js my-project-dir/js
            cp freesheet-web/dist/freesheet-all.css my-project-dir/css

Possible problems

  • If there are permissions errors from npm install, either use sudo or check this Stackoverflow question NPM throws error without sudo.

Using FreeSheet in a web page

Include the js and css files from the build in the head of your page eg: 

            <link rel="stylesheet" href="/css/freesheet-all.css"/>
            <script src="/js/freesheet-all.js"></script>

You will also need to include jQuery.

Get a reference to the FreeSheet object:

            <script>var FreeSheet = require('freesheet-web')</script>

The examples directory of the Freesheet Web repo is the best place to see what you can do with FreeSheet. More documentation coming...
